ABOUT

uwu-VS is a semi-modular video synthesis instrument for web browsers.

Inspired by ojack's live coding environment HYDRA, I wanted to make a browser app flexible enough for live performance but accessible to users without any prior video synth experience (no coding required, but MIDI scripting is encouraged :)

Built using GLSL and three.js, the structure is as follows:

  • Our {drawShader} creates an initial texture based on the sliders in the {WAVES} table
  • This texture is passed to our {feedbackShader} and processed based on the sliders in the {FEEDBACK} table

The active {drawShader} can be changed via the {SHADERS} table or keys [z], [x], [c]. Changing the active shader will not affect the value of the sliders.

To save a preset, press the [s] key. This will save the current arrangement of sliders to number keys [1]-[0]. Recalling a preset will update the slider values, but will not change the active shader. Once all ten slots are filled, saving a preset will overwrite the current selection.

All parameters can be controlled via the Web MIDI API (click here for supported browsers) meaning uwu-VS plays well with others! Try using a control surface or by sending MIDI messages from another program (Ableton, Max/MSP, etc.)
