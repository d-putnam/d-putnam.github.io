SVG gradient-mkr is a JavaScript web app designed to simplify the process of creating SVG gradients.

The user can manipulate the gradient via the UI, and the changes will be reflected in the page background.

The "show code" button will reveal the inline HTML producing the current image. Alternatively, the user can export the image to a static 4000 x 4000px .png (sans animation).
