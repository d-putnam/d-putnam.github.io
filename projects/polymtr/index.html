Polymtr is an algorithmic, polymetric drum sequencer that runs in your web browser.

Utilizing Tone.JS, all audio synthesis and processing occurs in real-time on the client machine.

Each voice is assigned an array of random values. Based on the threshold settings on each channel, this is translated to a pattern of notes. Raising the threshold to 100% will allow a steady stream of 16th notes.

Each voice has a variable pattern length, allowing for longer phrases with polymetric phasing.

For best performance, Google Chrome desktop is suggested.
