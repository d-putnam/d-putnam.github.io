# Hello!

I made this site to showcase my portfolio. 

The color scheme is generated on page load to provide a unique experience on each visit.

## Details

The background is comprised of a full-window inline SVG element with three colors.

On page load, three sets of RGB values in the range of 5-155 are generated for the background.

Two additional colors are generated by taking the largest and smallest values from the initial sets of R[] G[] and B[].

These colors are used at varied opacities to color the links, labels, hover effects, and overlays.

Hovering over my name in the navigation bar will produce an animation showing all five colors.
